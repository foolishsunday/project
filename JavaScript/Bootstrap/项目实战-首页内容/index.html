关于酒-水混合物有趣的是：研究者在八周的时间中，不断增加着混合物中酒精的比例。因此虽然研究初期的混合物中酒精比例只有10%，但是到结束时比例达到了20%。而且即便所有动物在整个试验中都可以不受限制的喝水，但是在过程中还是发生了一些有意思的事。

多年来，科学家们一直在努力，企图创造出一种可以将精神活动转换成语音的系统，而现在，加州大学旧金山分校的一组研究人员最终攻克了这一系统。 虽然这项技术仍需要一些微调，但总有一天它可以用于人工恢复那些因脑损伤、中风或帕金森氏症等神经退行性疾病而失去说话能力的人的声音。
